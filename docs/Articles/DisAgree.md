# “不同意”的故事

山本五十六，苏格拉底，岳飞不如咱家的娃。

山本五十六不同意。

二战前，日本海军元帅山本五十六就知道日本是打不过美国的。他曾是日本驻美大使馆的武官，亲眼亲身体验了美国工业和资源的庞大体量。他不同意对美作战。

虽然他极力反对对美作战，但军国主义内阁最终还是决定南下，夺取东南亚的资源，这就必须要对英美开战。

山本五十六不同意，但还是接受了上面的决定，在这种明知打不过还要打的情况下，只好进行大胆偷袭。偷袭珍珠港。

很快日本在太平洋节节败退，山本也绝望的被击落，坠入太平洋。

苏格拉底不同意。

公元前的古希腊，雅典公民苏格拉底总在广场上和人抬杠。通过不断追问，来揭露人们其实不是那么聪明，不是那么知道。

这惹恼了很多人，因为很多雅典公民的孩子学苏格拉底去追问自己父母，怀疑神权和传统，尤其开始批判雅典的民主制度。

雅典公民用民主投票判决苏格拉底死刑。他的学生前去营救，但苏格拉底并没有选择逃亡。

他要遵守雅典民主制度给他的判决，因为他是雅典公民，即使他不同意雅典的民主制度。

岳飞不同意。

南宋的民族英雄岳飞正在北伐，形势大好。举例开封只有45公里，敌人金兵是士气崩溃。此时皇帝连发十二道金牌召唤他撤军回朝。

岳飞不同意此时撤军。仰天长叹：“十年之功，废于一旦！所的州郡，一朝全休！”

然而岳飞还是遵守了他不同意的诏令，撤军回朝。从此南宋再也没有收复中原的机会了。

不久后，岳飞也被秦桧诬陷而死。死在了他严格遵守的制度中。

学生小孩不同意。

他不同意放下手上的玩具去写作业。

但受于家长强力的干预和鞭策，他只能去写作业。

他心里不服，他就是对抗，不配合，消极怠工。

他竟然能将自己的不同意坚持到底。他才是真的不同意。
