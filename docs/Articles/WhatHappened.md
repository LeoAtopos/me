# 何事发生

意义来源于叙事。

一个人认为何事发生决定了他的叙事，他在其中解读出的意义，进而决定了他的三观、思考方式和行动。

这里的关键词是“认为”，也就是他的主观如何认定。这就是唯心主义的思想根源。

而唯物主义的起源是首先认为是主观认定是不靠谱的，我们很难知道真相，很难知道全部，而全部真相——也就是真实上到底发生了什么——是与你的认为无关的在进行和发生的。所以，我们必然要小心的多方验证事实，排除人的主观因素来不断更新”何事发生“。

然而，当他人的主观认定成了自己的客观事实的时候，例如股票交易市场，唯心不能忽略他心，唯物不能理解他心，对于发生了什么的解释力都失效了。这就是现实，不同于唯心的内心世界，也不同于唯物的客观世界的现实世界。

现实世界最为复杂。

人们用经济学，社会学，政治学，人类学，文化和意识形态等不同的角度去分析现实世界，已经是很努力的在尝试了，但绝不会像文学之于内心世界，物理学之于客观世界那样有统治力。

不得不说，正常情况下，一个人学什么专精，就成了什么锤子，会发生看什么都是某种钉子的效应。这对于何事发生就产生了偏斜解读。

所以，往往是交叉学习了很多不同角度的人，猜得到全面，同时也是根本的理解。摸遍大象的全身。

摸遍大象全身，是很难的，需要太多的时间，花费太多的精力了。事实上每时每刻，人们都需要认定何事发生。

人们通常需要的是一个简化的认知模型，内化的感受力，去处理日常事务。类似一种快捷方式。

在没有意识到这点之前，这种认知模型就来自于周围环境的影响。所谓近朱者赤。

在意识到这点之后，人才可能自主的去梳理自己的认知模型，得到一些自己认定的”道理“。

这些道理有时候市侩，有时候超越，有时候自私，有时候天下为公，成了人们各自的人生哲学。

怀着各自不同的，很多时候也是不稳定的哲学而互相影响冲突的状况，就是现实世界正在发生的事情。

认定了如此的叙事，会产生什么样的意义呢？

我很难从如此广泛抽象的叙事中获得个人的意义解读。客观世界，现实世界的叙事，只能作为内心世界叙事的条件，背景和边界。

我个人的经历，自我的叙事是需要与客观世界叙事和现实世界叙事协调的。

经过另两个世界的叙事的校准，自我叙事才是有机的，不是孤立的，才是走出自我封闭的叙事。

探索世界，关心他人的意义大概就在于此了。

记日记整理自己的个人叙事的意义，也不言而喻了。
