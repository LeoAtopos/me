# 主体人与工具人

我们都不想当工具人，我们都要唤醒自己的主体性。但当人们共同协作做事的时候，难免就会分出主体人和工具人。

你在公司打拼多年，终于拿下一个大项目，主导这个大项目大干一场，以主人翁的精神投入工作，临到即将完成的时候，公司上头决定放弃这个项目，全体转投另一个赛道。甚至没和你提前沟通过。

一个知识博主在视频和直播中大谈人的主体性，不断的在以自己举例，告诉观众他是如何六经注我，专注于自己的事。观众看不到给他架机器的小哥，给他订机酒的小妹，为他审稿的阿姨跟他讨论的时候如何吃瘪，为他临事起意而被迫调整自己私人生活时间安排的工作团队。都是工具人。

带娃的时候，真的是父母在带娃吗？不，带娃的主体人是妈妈，其他所有人都是工具人。决定娃现在出门玩到底要不要带着帽子的最终拍板人是妈妈。只有妈妈放权了，爸爸才有决定权。至少我家是这样的。

一个事情的最终决定权，否决权是谁的意志决定的，在这件事上，这个人就是主体人，其他人都是工具人。

所以，历史总是写皇帝，权臣，大将，地方豪强。因为历史的走向绝大多数时候不是某个农民或士兵的意志决定的。就算有，那最终也是要反应到某个领导人的个人身上的。虽然说历史是人民创造的，但历史从来不是一个单独的人民创造，而是某个人民主导的。历史的创造总是来自某个主体人。

在所有事情上，分清谁是主体人，谁是工具人，自己是主体人还是工具人，就显得至关重要了。所谓阴阳。天行健，自强不息是阳，地势坤，厚德载物是阴。这是古老的智慧，也告诉我们工具人也有自己的高低标准。

我很想说康德的原话是，人不只是手段，人是目的本身。这句话被极端化传播成了，人是目的，不是手段。看出差别了么？康德并不排斥人作为手段，作为工具，他说的是“不只是”。

有时候，有些事上，我是主体人，要负责，要自强不息，另一些时候，另一些事上，我就是工具人，要接住，要厚德载物。人生就是在主体人和工具人之间来回切换，所谓一阴一阳之谓道。

但糟糕的是当工具人久了，无法切换到主体人，这会让自己很难受，作为生命的人，内心深处还是有原始的自主性欲望的。这就压抑了。

另一种是当主体人久了，坚决不当工具人，这会让别人很难受，在不该自作主张的事情上和真正的主体人发生冲突。这就暴躁了。

当然还有没分清楚自己是主体人还是工具人的情况，也挺糟糕的。以为自己是主体人，但被真正的主体人剥夺主体性的时候，那叫一个疼痛。以为自己是工具人，而错过了该做判断和决策的机会，那叫一个悔恨。

我想最好的关系像两个人即兴的共舞。如同对话一般的共同享受此刻，双方的主体性都得到了体现，在这件美事中，缺了谁的主体性，都不完美。但这太难，机缘有限。

我们也可以说，主体人和工具人的身份是叙事敏感的，换了不同的上下文，身份就可能互换。

在公司往哪个方向发展这件事情上，董事是主体人，经理是工具人。在公司具体要做什么项目，如何做这件事情上，经理是主体人，员工是工具人。在我该如何提升自我，赚钱这件事情上，员工是主体人，具体这个公司还是那个公司是这件事的工具人。董事也是工具人，资本是使用他们的主体人。他们被非人的主体支配。

我到底想说什么。可能也就更加清晰的看懂工具人和主体性这两个概念吧。看似矛盾的概念，是动态互补互换的，集体做事就是会如此分化才行。接受这种现实，找到找准自己的定位，才能避免之前提到的种种问题吧。
