# 现在的孩子玩的不一样了

我都四十岁了，依然还和孩子们在广场上玩。或者说，我爱和别人像孩子一样的玩。所以，带娃出去玩并不是我的家长义务劳动，而是我也可以享受的有趣活动。直到我感受到了一种恶心。

现在的孩子玩起来感觉不一样了。

今天在带娃的时候，广场旁边有个篮球架，一个年轻人，大概也就是二十左右在打篮球，就是投投球。我很自然的凑了上去，同时也有两个大概十一二岁的孩子拿这个4号的小篮球上来投篮玩。

前几球我还挺没觉得什么。我站在篮下，球下来在我附近，我就捞过来投一个。我想去公共球场打过野球的应该会觉得这很自然。但我渐渐注意到，这两波自己带球的人，绝不会投对方的球。

如果对方的球到了自己附近，偶尔会顺手给传回去，更多的时候是忽略，只玩只管自己的球。这让我有点尴尬了。

更让我惊奇的是，那个二十多岁的娃开始冲过来，拿本来会落到我附近的球。发生几次这样的情况后，我就走了。也许人家着急练球，不想分享。

我就走到远处陪娃玩。可没过一会儿，我再看向篮球场，现在有4个人，4个球。再仔细看会儿，你会发现，真的就是各投各的，各捡各的球。

没人互相说话。

这让我悲伤。

或许是我过度敏感了，或许我只是碰巧碰上了特例。但这种不舒服已经很久了，不只是这一次。

上一次这种感受是和楼下的小鬼们玩豆沙包。

丢沙包多简单的规则啊，谁被砸到就换谁来砸，空接沙包就加条命。可是现在的小鬼给加了许多规则。

沙包滚地了就不能算击中。

十秒内必须丢沙包。

不是谁被砸到就换谁来砸，而是下场等待。

空接了沙包可以救下场的人。

丢沙包的人空接了沙包可以喊「定」，躲沙包的不脚不移动。

我不知道还有什么我不清楚的规则，我猜肯定还有。

其中我最不认可的是，沙包打中脸就不算打中！

然后，可想而知，规则多了，就需要很多判断，很多判断到底发生没有就开始争执。

滚地了么？十秒到了么？空接了救谁呢？「定」喊完了你是不是动了，刚才的位置在哪里……

我只玩了5分钟，就开始厌烦他们了，根本没在玩游戏，全在嘴炮规则和判断。

尤其是其中一些看上去挺老爷们儿的男孩子，在哪斤斤计较，那样子我真想给他两拳，让唐僧闭嘴的那种拳。

我就感觉我们小时候那样的玩耍，真的是在玩，是在享受玩游戏。而他们是在努力追求自己能赢，享受能赢下游戏的努力。

那些篮球场的小孩子们真的是在打篮球，是在投篮，不是在玩篮球。

我不知道怎么会变成这样。那种朴素的「一起玩」去哪了？

游乐场，甚至家门口楼下都好难看到不认识的孩子稀里糊涂自然而然的就玩到一起去了的场景。

我的娃也是，也不会去认识陌生娃一起玩。可能他也有过和我一样的经历，去和人打篮球，会感到冒犯到别人的篮球了。

我反思下，是不是每一代人都会觉得下一代人怎么变了。是不是时代改变，这样的变化其实是很自然的？

或许都是吧。

那我真的是就是不适应现在这个时代了。该的。
