# 谈谈贾木许

贾木许是个1980年代开始创作的独立电影导演。听上去姓贾，但他是美国人，全名是Jim Jarmusch。

我第一次看到他的电影非常偶然。那是一部叫做《帕特森》的文艺片。我已经忘了是怎么点开他去看的，但看完真的令我感叹，哇，原来这就是文艺片啊。

《帕特森》的故事很简单。一个名叫帕特森的公交司机，在名叫帕特森的城市里工作，生活，写诗，没别的了。没有高潮，没有冲突。

帕特森只是如常的生活，周一到周日，碰到一些人，发生一些不咸不淡的事。只是在其中穿插着帕特森写的诗。

感叹文艺味道的醇厚之后，我只是记住了这个导演，没有深入他的其他影片，以后想要文艺的时候来取材就是了。多年以后的，我带着一个问题重新翻看他的所有影片，才有了现在想要书写的东西。

个人表达和大众喜好的冲突该如何解决？

贾木许告诉我，你可以拍大众喜欢的类型片，但你总能在其中找到个人表达的点，做出属于你自己的类型片。

单看《帕特森》是不会得到这样的答案的，因为他太文艺了，你甚至看不出他的类型。多看一些其他片子，我才发现，贾木许也是什么火就拍什么类型啊。

《不法之徒》是越狱片。《神秘列车》是在玩非线性叙事。《离魂异客》是西部片。《鬼狗杀手》是黑帮片。《破碎之花》是中年危机。《唯爱永生》是吸血鬼题材。《丧尸未失》是丧尸题材。《家庭日》是亲子关系。

这些很明显都是类型化的影片，是流行的，人们有认知的类型。但贾木许的丧尸不只啃食活人，还会不断重复他们生前的执念，热爱咖啡的丧尸会一壶一壶的往自己脸上倒咖啡，打网球的丧尸女孩会随便抓起什么反复挥动。贾木许的黑人杀手会遵循武士守则，我怀疑混沌武士对此是有借鉴的。

《帕特森》其实是爱情片。

在所有的讨大众喜好的类型化题材中，贾木许总能让你感受到他自己的强烈的个人风格，也就是他自己的个人表达。

1. 贾木许的镜头冷静克制。
2. 总会看到东方元素，日本或中国的。
3. 他总写陌生人的相遇。他们闲谈。
4. 他总会在多个故事里放入相同的话语或事物进行互文。
5. 他的角色大多情绪稳定，不会西斯底里。
6. 剧情中总有极其轻易的部分。如轻易偷到敞篷车，轻易获得了一笔财富或爱情，轻易潜入敌人堡垒。
7. 他总是有绝品的配乐。
8. 绝不给什么结局大反转或神展开。

在其他导演都在焦虑的往画面里塞信息点的时候，贾木许的影片允许你仔细观察，有时间琢磨一下。

贾木许喜欢的和擅长的其实从来没变。他的第一部长篇《长假漫漫》确实不是任何成熟的类型片，但已经有了后来贾木许的所有影片的个性特质。人的流浪，遇见一些人，发生一些对话，没有结局的结束。

所以，不要觉得个人表达和大众喜好是冲突的。沟通是一半一半的事情。不能全然不顾对方的喜好，只顾自我的表达。

人们喜欢的是熟悉加意外。贾木许先用类型片给了观众一些熟悉的预期，就可以在讲述的过程中放入自己的东西，成了一种别样的风味。

如果熟悉贾木许风格的人，反过来也会觉得他的侦探片会很意外。

如果你所表达的东西深刻又普世，理论上他是能融入到任何类型或题材中去的。

说到最后，让我推荐他的《地球之夜》吧。几个小故事不知为何，放在一起就很有味道。无法言说的意味，才是艺术最精华的内容啊。
