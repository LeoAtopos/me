# 如是我闻

原来佛经的开头会说“如是我闻”，这可太震撼了，这里有大学问，会让人爱上佛学。

有时候我多想给《论语》的每一篇前面都加上一句，如是有记。我们知道，论语是孔子的弟子和弟子的弟子写的，是回忆记录了孔子的一些言行的集子。我们确实能够通过它了解到一些孔子的行动和思想，但那是孔子的全部吗，有没有可能有遗漏和记错，或者理解偏误？

多少人读论语的时候，就是带着“啊，这就是孔子”的心理前提来读的？

佛经最初也是由佛陀的弟子和弟子的弟子集结在一起创作的。与佛陀最亲近的弟子阿难口述佛陀的教导，其他人一起监督认定。这可能就是为什么一定要加一句如是我闻在现场情景下的理由。这里可以看到佛学的较真。

这样的做法真的是佛家独门的。任何其他宗教经典，都多少带着一种不可置疑的气质，满篇都透露着“就是这样”的精炼的肯定感。

但佛经的如是我闻开头，就会把我立刻代入到它的创作现场，毫无遮拦回避的告诉我，我现在看到的佛经是怎么来的，这里的真相只是弟子们都认定阿难回忆的教导是确有其事。至于佛陀的本意，佛陀的全部思想，是否全部在这个佛经的文字里，那是每个人自己的判断了。

代言者表明了代言者的身份，阅读者也就能摆正阅读的态度。阅读只是提取信息，而非直接安装程序。这让我想起戏剧理论中的布莱希特体系，他要求演员时刻要提醒自己和观众，这是在演戏，不要过度代入剧中角色，戏剧不是让你沉浸，而是让你思考的，保留你自己的判断。

如今，布莱希特体系也是小众的。人们评判好的电影，戏剧，游戏，第一条就是很有沉浸感。如是我闻的传统在后来的佛学经典中也慢慢消失，心经读起来也是那么有沉浸感，佛学变成了佛教，越来越权威化了。

所以，在没有创作者再好心提醒读者事实真相之后，读者和观众只能自我提醒。

“尽信书，不如无书”是类似的提醒。“物自体不可知”是类似的提醒。但我更想时刻给自己一个通用提醒：如是我见。我所见的，我所读的，甚至我所经历的，并非事情的全部真相。因而这篇文章也只是：如是我思，而已。
