#  活在虚拟层很糟糕

都说现代社会，人被原子化。可是人还是要与其他人互动，否则就会孤独，偏执，焦虑不安。这就让现代社会提供了大量的虚拟互动产品，来解决这层需求。从而造就了一批只生活在虚拟层的人。

人有三层关系，由内而外是自我关系，私人关系，公共关系。一个人在私人关系中受挫，就会遁逃于自我关系中，以自我为中心，再从公共关系中找补私人关系缺失带来的社会存在空虚。

追偶像，打游戏，大篇写网评，读哲学，看剧看番，cos成另一个人去漫展，以某某铁粉自称，模仿者某种朋友圈中看来的生活方式，旅行，日杂或摇滚乐。没有长期稳定的私人关系。

当我们认为这就是大社会，现代社会该有的样子的时候，有没有想过，其实那都是虚拟的部落。其实，社会从来都是由私人关系网牢固的组成的，这个私人关系网才是社会的本体。

一切虚拟层的精神与社交活动，都是大众传媒产业的市场而已。

B站的徐师傅采访了几百人，大多人，尤其是年轻人，都有过度虚拟化的生活。看不到他们与真实身边人的健康关系（虽然大部分的时候都不是他们自己的原因），他们总是在谈某种理想的人生，追求什么美好状态，特别遥远，特别虚拟。

我曾经也是的。

我会告诉自己，身边的关系和人都太小了，太狭隘了，我要追求伟大的社会影响力，创造伟大传世的作品。却不知，自己的作品缺少的就是亲身的活人味，全是虚头巴脑的概念和模型。

我曾经认为虚拟层，文明，文化那一层才是人类社会更长久更真实的存在，但我发现这是个有病的想法。

写代码的人中有一句话，叫做“没有什么问题不是增加一个虚拟层解决不了的。”

虚拟层其实是为了解决问题而产生的工具。好么，现在却成了逃避问题的去处。

我们理应通过虚拟层来解决私人关系这一层的问题。私人社会网络才是最真实最实在最鲜活的社会本体。

用同好去结交更多的私人关系，分享哲学谈话解决个体困境，共同旅行做事共创美好回忆。

所见所爱乃具体某人，被爱被思的也是具体的自己。

你在公众视野中看到的名人和强人，也别忘了，他们的私人关系网络可能更强更精彩，只是不为人所知。

要相信，你最美好的人生，一定是某个很私密的个人关系中，而非某个聚光灯下。
