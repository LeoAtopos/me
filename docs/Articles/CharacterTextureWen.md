# 人以载文

我还很恐慌，因为标题这句人以载文的发现。

不知所措。写到哪算哪吧。

反思自己的创作，都是有感于某种智识性的新知的趣味。一个概念，一个情景，结构或认识上是新颖的，开悟的，于是就开始创作，于是就内容难产，于是就干巴巴的。为什么？

原来缺少了人味儿，活人味儿。

我总是急切的，想要最高效率的把我探索发现的新认知新结构表达出来，但其实最直接的表达就是把这些信息说出来，完全没有必要去包装成一个故事，故事性的文章或游戏。强行如此的话，就产生了干巴巴的工具人，或者直接难产于内容。

沟通，表演，变成了铺排和说教。对，这是教育，不是人文，是文明，而非文艺，更非文娱。

如果只是看到文以载道的传统，那干巴巴的教育说教是没问题的。然而，我想要的并非学术，而是更大众的影响力。

这就要还原人与人之间的文的交流的本质了，八卦。八卦另一个人的故事。

人天生对他人感兴趣，虽然底层都是为了自己更好的做人，不过只关注自己是难成长的。

人和人要进行文的交流以载道，那文本身是由什么承载的呢？

在我开始以人（或者说人物）的视角再去看所有的作品和创作思路的时候，我发现，一切都顺了。

人以载文。

这个人可以是自己，如果我坦诚的表达暴露自己。

这个人可以是虚构的人物，我将想要表达的智识认知的文装载到这个人物上，或者反向从这个人身上剥夺走，就可以凸显我要表达的文。

这个人可以某个现实人物，公众的或还没有被公众认识的，在他的身上能看到你所想要关注和强调的文的作用。

有了人，就有了生命，有了生命力，他就要产生欲望，面对困难或挑战，就要追求逃避或行动，就有了冲突和经验的潜力。人们就会自然的关注，我也能自然的生成新的内容。

先不说这样的内容是否质量上乘优质，至少这样从人出发的创作思路是能够自然生成内容的，并保证结构上符合八卦天性，能够引发关注的。

或者说质量好坏，还是看作者自己是什么水准的人，那自然会体现在文中。这就不是思路或方法论能解决的问题了。

人以载文的思路，他只解决如何自然生成文的问题。

事实上，我所喜欢的北野武，昆汀，时空旅人，周星驰，艾迪芬奇，都是人的文的高手，是生命力审美的。

我之所以忽略了这点，是因为另一个我的存在，一个因为现实的人际受挫而遁逃于智识认知以维系自我优越感叙事的我。

遁逃入了纯文的世界。这也是现代社会的普遍现状，深刻问题之一。理念之争，利益计算，标签攀比，虚拟崇拜。

而本能上，人们还是关心具体的人的，人们还是关注人的故事的，是要展现和共鸣于生命力的，原子化的现代个体又无法获得持续深刻的真人存境，只能在虚拟的文（尤其是网络世界）中如原始人般的鲁莽冲突，来感受自己的如人般的存在。

扯远了。还是说回创作吧。

文章要表达观点，可以说人的故事去举例引申。

小说故事要有意思，就得从“角色爱”下手，先构想一个表达我的文的让我感兴趣或深爱的角色。这就能理解很多作家所说的，角色自己在表演故事是什么意思了。

创作游戏，更要从角色入手来创造内容，引申机制。小丑牌的小丑，吸血鬼幸存者的吸血鬼和幸存者，艾迪芬奇的家族成员，无不如此。

人以载文，是创作的基本规律，是受众接受文的自然通道。
