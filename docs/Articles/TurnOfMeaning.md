# 意义的结构转向(Turn Of Meaning)

为什么当你寻找到人生的意义，它马上又变化了呢？这是曾经在某一本寻找人生意义的书中看到的一句话。我记得这本书最后的结论是寻找意义，就是人生的意义。你总是在寻找一个更深的更远的意义当中过自己的一生。我并不满足这样的答案。

我的发现是大家寻找意义的某种前提是错的，至少是不充分的。

所有人在寻找意义的这件事上，都是在向前看。他们假想在未来的一种状态，然后构建一套叙事去达到这个状态，从这样的一种向前看的叙事中获得意义感。他们寻找意义的起点是存在主义的那一句，我被抛掷在世界上，存在先于意义。也就是说，他的存在，他的过去是没有意义的，所以才要向未来去寻找。

“今天，妈妈死了。或者昨天，我不知道。”加缪的《局外人》的这种开场，就是一种典型的将个人过往的经历进行切割的做法。他们首先就认为存在本身是一种荒谬的状态。我相信任何儒家和道家都不能接受这种前提。

通俗一点的看法是，我要赚到多少多少钱，我要达到什么样的人生体验，我要达到什么样的生活标准，如此如此，我才真的幸福。前天下之忧而忧，后天下之乐而乐，或者我要成为对他人有用的人，我要尽可能地帮助他人，推进文明进步，其实和这些通俗的观点的逻辑是一样的，是一种状态和效用的逻辑。

至少你可以发现，所谓的意义都是在某种你坚信的叙事当中产生的。你如果不经历很多事情，并在中间找到最重要的一些基础的信念，由他们组成一个你认同的叙事，那你是很难凭空产生一种意义感的。

受过穷的人要钱。社会关系断裂的原子人因虚无而寻求自己的主体性。曾经因一些奇怪问题的思考和解答感到过过开悟的人，他就追求对人生全面而根本的认知。但这些叙事中都忘记了一件最根本的事情。那就是我们是人，人是生命。

想想看，我们要确立一个人生意义的下一步是什么？那答案是用这个人身意义作为指导目标，去指导现在的行动。去判断现在该做什么，该怎样做，该不做什么，该如何投入资源和时间。

当你的意义建立在未来的某个状态时，它就会产生两种情况。一种是这个状态是容易达到的那你就要进一步去寻找后面的人生意义是什么了。另一种状态是这个目标很难达到。或者当你在确立目标的时候，他就是永远无法达到的。比如说伟大的革命理想什么的。

第一种状态是短暂的人生意义，所以他并不终极。第二种状态是类似于一种逃避，我只要把我的意义目标做的特别大，那我就不用再面对达成之后还有什么意义的这样的问题。

然而，我们不是很清晰地能够知道一个当下的行动，它是之前的状态的函数吗？我们现在做什么难道不应该源自于我们当下是什么，之前发生了什么吗？也就是意义的叙事，应该从对未来的某种状态的追求转向过去自我的认识。

其实我们非常善于做这样的事情，例如我是一个父亲，所以我要怎样怎样，我是一个教师，所以我要怎样怎样。我们总是能够非常确定过去发生的事情和当下的自我的状态。那么从这里就可以推出自己要去做什么，不是吗？

而当我们形而上的去问人生的意义是什么的时候，我们显然需要脱离父亲教师这样的身份，脱离具体的情境和环境。具体的人的历史在这样的概括性叙事当中，唯一能确认的就是我们每个人都是人，我们每个人都是一个具体的生命。

此时，我们观察这个世界上的生命，你会发现所有的生命都在追求成长，追求存续，追求繁荣。从热力学的角度去看，就是追求最大的负熵。

所以我们是不是可以不要把人生的意义放在未来的某个状态的实现，什么立言立功立德，追求不朽，追求主体性的实现，追求人生的体验等等这样的在未来的某种状态。而是看见我们的过去和现在看见一种最根本的无法否认的无法回避的事实，那就是我们是一个生命。我们需要活着，我们需要自然的释放我们的生命力。

这样我们再看所有的所谓的意义，所谓的追求的目标，甚至说所有的美，那都是一种对生命力释放的追求、展现、交流和共鸣。

无论阶级是否存在，无论你在哪种文化，无论是小孩还是垂垂将死的老人，当我们认识到自己是一个生命，我们看见自己生命的存在的时候，我们自然的就是需要去做点什么，去释放这种生命力，去感受这种生命。

对生命力的审美，给予了一切活物存在的意义。

我这里并不是否认所谓终局思想的价值。但我认为那是一种工具理性，而不是一种本真的、全面的、自然的适用于讨论人生的思想。可以说，任何类似于终极思想的这种面向未来的思考，其实都是为了让当前这个生命体感受到生命力的存在和勃发。

这也是符合一种价值在人，在于生命的价值判断。

所以我们看到哲学三问第一句是我是谁。这个逻辑是对的，人就是可以概括的认为我是一个生命，我是人。同时对于具体的人来说，我就可以有我具体的身份，具体的历史。具体可靠的人生叙事。无论是宏大的概括的人，还是具体的、个体的人，都可以从已知的可靠的事实叙实中推演出自己的当前目标、未来目标。

至于在未来想要达到的某种状态，那都是一个对于当前自我叙事的某种延伸，这个延伸未必可靠。所以甚至你不必执着于这样的人生目标。

如果你能明白土蝉爬上枝头，就是要大声的嘶吼，那么你也就能明白我今天所说的这种意义的结构转向是什么了。他不会考虑自己的嘶吼是为了什么，因为他这个生命的生理结构，就会让他去嘶吼去交配去繁衍，这是他的自然，而你我都在自然当中。
