# 麻辣烫不辣，忍住那口优越感

老婆说晚上吃麻辣烫，我问娃不吃辣啊，他说没事，虽然是麻辣烫，但上面写了不辣。有逻辑洁癖的人可能会受不了，但很多时候我也相信这种事情会发生，表达了下不可思议后，就没再多说。

结果晚上饭做到一半，老婆说，不行，这个麻辣烫的调料尝了下，还是辣的，然后向我找抱怨。

“我是不是早就怀疑过？麻辣烫不辣？”这样句马后炮，我没有说出口。字句在我脑子里成型，溜出了唇齿，又被我嗦了回去了。

管理好自己，不要输出负能量。口嗨爽一下，我也不会多快乐，但今晚的气氛肯定是被我污染了。

人总有一些事情，看了会嫉妒，听了就会生气，很多情商自律厉害的家伙都说，要管理好自己的负能量来源。

其实，人自己也非常可能是别人负能量的来源，自己作为负能量的出口，也要管理好啊。
